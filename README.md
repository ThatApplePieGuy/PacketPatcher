# PacketPatcher

Spigot plugin that fixes various client-server issues with 1.8

Requires [PacketEvents](https://github.com/retrooper/packetevents/)

## Fixes
> All fixes can be toggled individually within the config

- Fix ghost liquids when placing buckets
- Split oversized chat messages to avoid kicking players
- Fix projectile positions desyncing
- Make hearts display more accurately for players
- Fix a keepalive ping spoofing exploit
